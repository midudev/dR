dR

What's that?

A simple domReady library that let's you know when, obviously, the dom is ready in only 295bytes (minified).

Based on:

  • domReady
  • quoJS
  • Smallest DOMReady code, ever

Example of usage

Using JavaScript


    dR( function() {
     console.log( 'Yikes! Dom is ready, dude!' );
    } );

Using CoffeeScript


    dR ->
      console.log 'Yikes! Dom is ready, dude!'
